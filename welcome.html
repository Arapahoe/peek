Welcome to Peek Warp!

You have been updated to the latest version of the Peek extension, nicknamed 'Warp'. This update significantly improves performance, removes the maximum limit for previews, adds support for more file types, fixes lots of bugs, and more.

Getting Started

Peek shows previews over file download links. Just hover your mouse over a link to a file, and the preview will appear.

Try clicking here and hover over some results.

To open Peek settings, click the Peek icon in the toolbar. You can also click the button below.

Supported files

Peek can create previews for the following file types:

  • PDF, TXT, RTF documents
  • Word documents (.doc, .docx)
  • Excel spreadsheets (.xls, .xlsx)
  • PowerPoint files (.ppt, .pptx)
  • WebM, MP4, GIFV, GIFV, and OGG/OGV video
  • MP3 and WAV Audio
  • JPEG, PNG, GIF, SVG, PNG, APNG, ICO, and BMP images

Peek can also show documents hosted on Google Docs and Apple iCloud. However, iCloud support is limited to Keynote files, as Apple doesn't allow other documents to be embedded.
