Document viewer

Peek uses an external viewer to display Microsoft Word (.doc, docx), Excel (.xls, .xlsx), and PowerPoint (.ppt, .pptx) files. You can use the Google Docs or Microsoft Office Online viewer.

Note: This does not affect PDFs, RTFs, and other document types.
